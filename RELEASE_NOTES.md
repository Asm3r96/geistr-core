# Geistr Core 1.0.0

Initial public release of Geistr Core: a reusable desktop-first AI app foundation you can run as-is or build on top of.

## Included

- Electron + React desktop AI workspace.
- Pi SDK-backed agent runtime.
- Provider/model selection and provider auth foundation.
- Persistent local SQLite sessions.
- Streaming chat UI with thinking and tool activity visibility.
- Built-in provider-independent `web_search` and `web_fetch` tools.
- Memory indexing, retrieval, tools, and Memory Galaxy visualization.
- Skills and MCP support.
- Four permission modes: Read only, Request approval, Default, and Full access.
- Generic background loop runtime and session compaction loop.

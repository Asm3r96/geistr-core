# Changelog

## 1.0.0

Initial public release of Geistr Core.

Includes:

- Desktop AI workspace built with Electron, React, and TypeScript.
- Pi SDK-backed agent runtime.
- Provider/model selection and provider authentication foundation.
- Persistent local SQLite sessions.
- Streaming assistant responses with thinking and tool activity visibility.
- Built-in provider-independent `web_search` and `web_fetch` tools.
- Memory indexing, retrieval, read/write tools, and Memory Galaxy visualization.
- Skill catalog and read-only `skill_load` tool.
- MCP server support.
- Four access modes: Read only, Request approval, Default, and Full access.
- Generic background loop runtime with session compaction loop.
- Local desktop packaging flow.
